# Google Analytics

## Google Analytics

### Creating a Hostname Filter

1. `Admin > View > Filters` 로 이동하여 새 필터를 만듭니다.
2. `Custom`를 선택하고 "Include", "Hostname" 필드, 필터 패턴 *your-website-url*을 입력하십시오.
3. 사용할 수있는 웹 로그 분석 데이터가있는 경우 필터가 의도 한대로 작동하는지 확인할 수 있습니다
4. 저장하는 것을 잊지 마십시오!
